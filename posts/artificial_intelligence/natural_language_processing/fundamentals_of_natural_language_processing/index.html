‚ÄÉ‚ÄÉÂºè‰∏≠Ôºö \(C(w,c)\) Ë°®Á§∫ËØç \(w\) Âíå‰∏ä‰∏ãÊñá \(c\) Âú®ËØ≠ÊñôÂ∫ì‰∏≠Âá∫Áé∞ÁöÑÊ¨°Êï∞Ôºà‰πüÊàê‰∏∫È¢ëÊ¨°ÔºâÔºõ \(\sum_{c'}C(w,c')\) ‰∏∫Ë°®1ÊåâË°åÊ±ÇÂíåÔºå \(\sum_{w'}C(w',c)\) ‰∏∫Ë°®1ÊåâÂàóÊ±ÇÂíåÔºõ \(\sum_{w'}\sum_{c'}C(w',c')\) ‰∏∫ÂÖ®ÈÉ®ÂÖ±Áé∞È¢ëÊ¨°ÁöÑÂíå„ÄÇ‰ª£ÂÖ•‰ª•‰∏ä \(3\) ‰∏™ÂÖ¨ÂºèÔºåÂºè(2)ÂèØËøõ‰∏ÄÊ≠•ÂÜô‰∏∫

\[ \begin{aligned} PMI(w,c)=&\log_2\frac{P(w,c)}{P(w)P(c)}\\[15pt] =&\log_2\frac{\displaystyle\frac{C(w,c)}{\sum_{w',c'}C(w',c')}}{\displaystyle\frac{\sum_{c'}C(w,c')}{\sum_{w'}\sum_{c'}C(w',c')}\cdot\frac{\sum_{w'}C(w',c)}{\sum_{w'}\sum_{c'}C(w',c')}}\\[30pt] =&\log_2\frac{\displaystyle C(w,c)}{\displaystyle\frac{\sum_{c'}C(w,c')\sum_{w'}C(w',c)}{\sum_{w'}\sum_{c'}C(w',c')}} \end{aligned} \tag{4} \]

‚ÄÉ‚ÄÉÂè¶Â§ñÔºåÂΩìÊüê‰∏™ËØç‰∏é‰∏ä‰∏ãÊñá‰πãÈó¥ÂÖ±Áé∞Ê¨°Êï∞ËæÉ‰ΩéÊó∂ÔºåÂèØËÉΩ‰ºöÂæóÂà∞Ë¥üÁöÑPMIÂÄº„ÄÇËÄÉËôëÂà∞ËøôÁßçÊÉÖÂÜµ‰∏ãÁöÑPMI‰∏çÂ§™Á®≥ÂÆöÔºàÂÖ∑ÊúâËæÉÂ§ßÁöÑÊñπÂ∑ÆÔºâÔºåÂú®ÂÆûÈôÖÂ∫îÁî®‰∏≠ÈÄöÂ∏∏ÈááÁî®PPMIÔºàPositive PMIÔºâ‚ÄåÁöÑÂΩ¢ÂºèÔºåÂç≥Ôºö

\[ PPMI(w,c)=\max\set{PMI(w,c),0} \tag{5} \]

def pmi(M, positive=True):
    col_totals = M.sum(axis=0) #ÊåâÂàóÊ±ÇÂíå
    row_totals = M.sum(axis=1) #ÊåâË°åÊ±ÇÂíå
    total = col_totals.sum() #ÊÄªÈ¢ëÊ¨°
    expected = np.outer(row_totals, col_totals) / total #Ëé∑ÂæóÊØè‰∏™ÂÖÉÁ¥†ÁöÑÂàÜÂ≠ê
    M = M / expected #ÂèñÂØπÊï∞ÂâçÁöÑPMIÂÄº

    if positive:
        M[M < 0] = 0.0 #ÊâßË°åPPMIÁöÑmaxÊìç‰Ωú
    return M #PPMIÂÄº
